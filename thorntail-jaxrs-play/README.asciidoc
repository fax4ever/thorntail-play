# thorntail-jaxrs-play

Sample REST project, using Thorntail v4.

## Build demo
----
mvn clean install
----

## Run demo
----
./target/thorntail-jaxrs-play-1.0-SNAPSHOT-bin/bin/run.sh
----

## Load static content
----
http://localhost:8080/ciaociao.html
----

## Load REST resource
----
http://localhost:8080/rest/ciao
----
